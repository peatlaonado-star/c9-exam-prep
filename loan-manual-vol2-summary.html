1. วัตถุประสงค์และหลักเกณฑ์การกู้เงิน

ข้อ 6 — วัตถุประสงค์การกู้เงิน

📌 วัตถุประสงค์การกู้เงินของเกษตรกร ให้เป็นไปตาม ข้อ 15 แห่งข้อบังคับฉบับที่ 44
  • ธนาคารกำหนด อัตราส่วนขั้นต่ำแห่งการลงทุนของผู้กู้ ไม่น้อยกว่า ร้อยละ 20 แห่งค่าลงทุนตามแผนงานหรือโครงการ
  • กรณีที่อัตราส่วนขั้นต่ำฯ น้อยกว่าร้อยละ 20 → ให้อยู่ในอำนาจ ผู้ช่วยผู้จัดการสายงานบริหารงานสินเชื่อ

ข้อ 8 — การผ่อนผันราคาประเมินและวงเงินจำนอง

📌 กรณีผ่อนผันราคาประเมินและวงเงินจำนอง — ให้ลูกค้ากู้เงินมากกว่าร้อยละ 50 แต่ไม่เกินร้อยละ 80
  • ✅ อยู่ในอำนาจ ผู้จัดการสาขา สามารถดำเนินการแทนธนาคารได้
  • ✅ ผู้ช่วยผู้จัดการสาขาทำการแทนได้ โดยผู้จัดการสาต้องมอบหมายหน้าที่ให้เป็นลายลักษณ์อักษร

ข้อ 11 — การอนุมัติรายงานเบิกเงิน

📌 ผู้อนุมัติรายงานเบิกเงิน — ให้เป็นไปตามระเบียบที่กำหนด โดยผู้มีอำนาจตามระดับชั้น

ข้อ 12 — การตรวจงานตามแผนงาน

📌 การตรวจงานตามแผนงาน/โครงการที่ใช้เงินกู้ อยู่ในอำนาจของ:
  • ผู้จัดการสาขา
  • ผู้ช่วยผู้จัดการสาขา
  • หัวหน้าหน่วยอำเภอ
  • พนักงานพัฒนาธุรกิจ

ข้อ 15 (แห่งข้อบังคับฉบับที่ 44) — วัตถุประสงค์การให้เกษตรกรกู้เงิน

⭐ วัตถุประสงค์ 7 ประการ
  1. ค่าใช้จ่ายในการประกอบอาชีพ หรือฟื้นฟูการประกอบอาชีพ
  2. ค่าลงทุน — สินทรัพย์ที่มีอายุการใช้ > 1 ปี
  3. ชำระหนี้สินภายนอก — แก้ไขหนี้ที่มีดอกเบี้ยสูง
  4. พัฒนาความรู้ — อบรม สัมมนา ดูงาน ฝึกงาน
  5. พัฒนาคุณภาพชีวิต — การศึกษา รักษาพยาบาล ที่อยู่อาศัย
  6. รอการขายผลิตผล — ชะลอการขายเพื่อไม่ให้ถูกกดราคา
  7. ค่าใช้จ่าย/ค่าลงทุนตามโครงการ ที่ได้รับความเห็นชอบจากคณะกรรมการ

2. หลักการให้กู้เงิน

2.1 ตามลักษณะของการประกอบอาชีพ

  • 🌾 ทำนา, ทำไร่, ทำสวน — อาชีพเกษตรกรรมหลัก
  • 🐄 เลี้ยงสัตว์, ทำประมง — ปศุสัตว์และประมง
  • 🪱 เลี้ยงไหม, ทำนาเกลือ — อาชีพเฉพาะทาง
  • 🌲 ปลูกไม้สน, ปลูกป่า — อาชีพระยะยาว

2.2 ตามลักษณะของการใช้เงินกู้

💰 ค่าใช้จ่าย
เงินทุนหมุนเวียนในรอบปี (ผลิตรอบเดียวหรือหลายรอบ)
🏗️ ค่าลงทุน
สินทรัพย์ที่มีระยะเวลา > 1 ปี จึงจะให้ผลตอบแทนคุ้มค่า
🔄 ฟื้นฟูอาชีพ
เพิ่มรายได้ให้ลูกค้าที่ประสบปัญหาจนไม่สามารถชำระหนี้
📚 พัฒนาความรู้
อบรม สัมมนา ดูงาน หรือฝึกงานด้านเกษตร/อาชีพอื่น
🏡 พัฒนาคุณภาพชีวิต
การศึกษา รักษาพยาบาล ปรับปรุง/ซ่อมแซมที่อยู่อาศัย
💳 ชำระหนี้สินภายนอก
แก้หนี้ที่มีดอกเบี้ยสูง เสี่ยงสูญเสียที่ดินทำกิน
📦 รอการขายผลิตผล
ชะลอการขายผลิตผลเกษตรที่ผลิตพร้อมกันจำนวนมาก
📋 ค่าใช้จ่าย/ลงทุนตามโครงการ
โดยได้รับความเห็นชอบจากคณะกรรมการ

2.3 หลักเกณฑ์วิเคราะห์ตัวผู้กู้ (5Cs + เพิ่มเติม)

🧑‍🌾 ปัจจัยที่ต้องวิเคราะห์:
💰 ฐานะทางการเงิน
⭐ ชื่อเสียง ความซื่อสัตย์สุจริต
📊 ประวัติการใช้/ชำระหนี้
🎓 ความรู้/ประสบการณ์เกี่ยวกับอาชีพ
📋 ความต้องการเงินกู้ตามแผนงาน
💵 รายได้
✅ ความสามารถชำระหนี้
🏠 หลักประกันเงินกู้

3. ขั้นตอนกระบวนการให้กู้เงินและเงื่อนไข (6 ขั้นตอน)

⏱️ กระบวนการ 6 ขั้นตอนตามข้อบังคับฉบับที่ 44 และ ฉ.45
1 รับแจ้ง + ชี้แจง → 2 สอบ สืบ ปรับปรุง → 3 วิเคราะห์เงินกู้ → 4 ตรวจทาน + ให้ความเห็น → 5 พิจารณาอนุมัติ → 6 แจ้งผล + จ่ายเงิน

ขั้นตอนที่ 1 — รับแจ้งความประสงค์ / ชี้แจงหลักเกณฑ์

  • 🔍 ตรวจสอบคุณสมบัติเบื้องต้น — สถานะบุคคลล้มละลาย, บุคคลเฝ้าระวัง, ประวัติการเงิน
  • 📢 ชี้แจงหลักเกณฑ์ เงื่อนไข และขั้นตอนให้ลูกค้าเข้าใจครบถ้วน

ขั้นตอนที่ 2 — สอบ สืบ ปรับปรุงข้อมูล

  • 📝 สอบถามวัตถุประสงค์ จำนวนเงินกู้ ทรัพย์สิน หนี้สิน รายได้ รายจ่าย
  • 🔄 ปรับปรุงข้อมูลลูกค้าให้เป็นปัจจุบัน — บันทึกใบคำขอสินเชื่อ + จัดทำเอกสารประกอบ
  • 📊 สืบค้นรายงานข้อมูลเครดิต — ตรวจประวัติชำระหนี้ ภาระหนี้ ความสามารถก่อหนี้

ขั้นตอนที่ 3 — วิเคราะห์เงินกู้

แบ่งตามวงเงินกู้ 3 ระดับ (ดูรายละเอียดใน หัวข้อ 4)

  • 💰 ≤ 500,000 บาท → Mini CAP / W/C
  • 💰 500,001 – 3,000,000 บาท → AGRI CAP Branch / SMAE CAP Branch
  • 💰 > 3,000,000 บาท → โครงการละเอียด + อัตราส่วนทางการเงิน

ขั้นตอนที่ 4 — ตรวจทานและให้ความเห็น

ตรวจสอบความถูกต้องครบถ้วนของเอกสาร + วิเคราะห์กลั่นกรองข้อมูล

ระดับวงเงินผู้มีอำนาจ
≤ 500,000 บาทหัวหน้าหน่วยอำเภอ
≤ 1,000,000 บาทผู้ช่วยผู้จัดการสาขา
≤ 3,000,000 บาทผู้จัดการสาขา

ขั้นตอนที่ 5 — พิจารณาอนุมัติ

อำนาจการอนุมัติตามระเบียบฉบับที่ 89 ว่าด้วยการพิจารณาอนุมัติสินเชื่อ

ขั้นตอนที่ 6 — แจ้งผลและจ่ายเงิน

  • 📨 พนักงานพัฒนาธุรกิจ (พพธ.) แจ้งผลอนุมัติและเงื่อนไขให้ลูกค้าทราบ
  • 📄 หนังสือกู้เงินที่อนุมัติแล้ว — รับเงินกู้ภายใน 3 เดือน นับจากวันอนุมัติ (ถ้าครบกำหนด → ประทับ "ยกเลิก")
  • 💀 กรณีลูกค้าถึงแก่ความตาย:
    • ก่อนวันนัดโอนเงิน → ยกเลิกการโอน ปรับปรุงบัญชี
    • ตั้งแต่วันที่ระบุโอน → ถือว่าถูกต้อง ไม่ต้องปรับปรุง
  • ⏰ กรณีสิ้นสุดสัญญา + ไม่มีหนี้คงเหลือ → ยกเลิกหนังสือกู้โดยปริยาย
  • 🔄 กรณีหนังสือกู้ฉบับเดิมสิ้นสุด ยกเลิก หรือเปลี่ยนวงเงิน → ดำเนินการตามระเบียบ

4. การวิเคราะห์เงินกู้ — 3 ระดับวงเงิน

ระดับที่ 1: วงเงินรวม ≤ 500,000 บาท

ประเภทสูตร/หลักเกณฑ์
🌾 เกษตร Mini CAP — ค่าใช้จ่ายในการผลิต แต่ไม่เกิน 70% ของรายได้ (ไม่นำค่าใช้จ่ายในครัวเรือนมากำหนดวงเงินกู้)
💼 อาชีพอื่น W/C = ลูกหนี้การค้า + สต็อก — เจ้าหนี้การค้า
ใช้อัตราส่วน: DSCR, ICR, LTV, WCR

ระดับที่ 2: วงเงิน > 500,000 บาท แต่ ≤ 3,000,000 บาท

ประเภทสูตร/หลักเกณฑ์
🌾 เกษตร AGRI CAP Branch — ประมาณการรายได้-ค่าใช้จ่ายใน 1 รอบการผลิต ตามข้อเท็จจริงของผู้ขอกู้
💼 อาชีพอื่น SMAE CAP Branch หรือ W/C = ลูกหนี้การค้า + สต็อก — เจ้าหนี้การค้า
อัตราส่วน: D/E, DSCR, ICR, LTV, WCR
  • ✅ กำหนดวงเงินกู้ได้ไม่เกิน 80% ของค่าลงทุน
  • 📊 วิเคราะห์ DSCR, ICR, LTV, D/E, LTD/E, NPV, IRR, B/C, PB
  • ⭐ หลักสำคัญ: พิจารณาความสามารถชำระหนี้เป็นหลัก
  • 💰 รายได้รวมทั้งหมด — ต้นทุนการผลิต — ค่าใช้จ่าย = ความสามารถชำระหนี้

ระดับที่ 3: วงเงิน > 3,000,000 บาท

📋 ต้องจัดทำเป็นโครงการโดยละเอียด
  • แสดงข้อมูลโดยละเอียดของแผนการลงทุน
  • ใช้เครื่องมือวิเคราะห์ทางการเงิน: NPV, IRR, B/C Ratio, Payback Period
  • ประเมินความเสี่ยงและผลตอบแทนของโครงการ

หลักเกณฑ์การวิเคราะห์โดยรวม

  • ✅ กำหนดวงเงินกู้ได้ไม่เกิน 80% ของมูลค่าการลงทุน (ตามข้อมูลที่พิสูจน์และเชื่อถือได้)
  • ✅ ได้ส่วนร่วมของผู้กู้ไม่น้อยกว่า 20% ของค่าลงทุน
  • ⭐ พิจารณาความสามารถชำระหนี้เป็นหลักสำคัญที่สุด

5. การกำหนดมาตรฐานผลผลิต + หนังสือกู้เงิน

5.1 การกำหนดมาตรฐานผลผลิตและมูลค่าผลิตภัณฑ์

วัตถุประสงค์
  1. 📊 เป็นเกณฑ์ในการประเมินรายได้และค่าใช้จ่ายในการผลิต
  2. 🌾 กำหนดมาตรฐานผลผลิตทางการเกษตรของแต่ละพื้นที่
  3. 🏦 เป็นเครื่องมืออำนวยสินเชื่อธนาคารที่มีมาตรฐาน
  4. 📉 ลดความเสี่ยงในการให้สินเชื่อธนาคาร
  5. 🛡️ ป้องกันความเสียหายที่จะเกิดขึ้นจากการประกอบอาชีพ
ขั้นตอนปฏิบัติงาน
รวบรวมข้อมูลการผลิตทางการเกษตรทั้งด้านพืชและสัตว์ของแต่ละพื้นที่ → กำหนดมาตรฐาน → ใช้เป็นเกณฑ์สินเชื่อ

5.2 หนังสือกู้เงิน — เครดิตเงินสด

5.2.1 เงินกู้เครดิตเงินสดแบบรับรองรับผิดอย่างลูกหนี้ร่วมกัน (คสก.)

  • วงเงินสูงสุด: ต่อรายไม่เกิน 200,000 บาท
  • ผู้กู้: ลูกค้าในกลุ่มเดียวกัน อย่างน้อย 5 คน ขึ้นไป
  • ลูกค้าแต่ละคน อาจเป็นผู้รับรองรับผิดในสัญญา คสก. มากกว่า 1 สัญญาได้
  • ระยะเวลาหนังสือกู้: ตามความเหมาะสม แต่ไม่เกิน 5 ปี นับแต่วันทำหนังสือกู้
  • ชำระคืนต้นเงินกู้: ภายในไม่เกิน 12 เดือน (มีเหตุพิเศษไม่เกิน 18 เดือน)

5.2.2 เงินกู้เครดิตเงินสดแบบรายคน (คสร.)

  • วงเงิน: ตามความเหมาะสมของค่าใช้จ่ายการผลิต
  • หลักประกัน: บุคคลค้ำประกัน / จำนอง / อื่น ๆ ตามธนาคารกำหนด
⚠️ ข้อควรจำ
  • จำนวนต้นเงินกู้ที่จะเบิก + ต้นเงินกู้คงเป็นหนี้ ต้องไม่เกินวงเงินกู้ที่ธนาคารกำหนด
  • กำหนดระยะเวลาหนังสือกู้ตามความเหมาะสม แต่ไม่เกิน 5 ปี

6. ประเภทเงินกู้พิเศษและพัฒนาคุณภาพชีวิต

6.1 เงินกู้เพื่อเป็นค่าลงทุนในการประกอบอาชีพเกษตรกรรม

📌 หลักเกณฑ์สำคัญ
  • 💰 ผู้กู้ต้องมีส่วนร่วมลงทุนเองไม่น้อยกว่า 20% ของค่าลงทุนทั้งหมด (กรณีกู้ > 50,000 บาท)
  • 📅 จ่ายเงินกู้เป็นรายงวดตามความก้าวหน้าโครงการ
  • 🚫 หากใช้เงินกู้ไม่ถูกต้องตามวัตถุประสงค์ → ระงับงวดถัดไป
  • 📢 แจ้งธนาคารหากจะลงทุนเพิ่ม (กระทบกระแสเงินสด)
  • ⏰ ระยะเวลาชำระหนี้ไม่เกิน 15 ปี (มีเหตุพิเศษไม่เกิน 20 ปี) — ปลอดต้นไม่เกิน 5 ปีแรก
  • 🏗️ ก่อสร้าง 100,000 - 500,000 บาท ต้องมีแบบแปลนคร่าว ๆ
  • 🏠 ใช้ที่ตั้งโครงการเป็นหลักประกันลำดับแรก (ยกเว้นมีเหตุผ่อนผัน)
🌳 กรณีที่ดินในเขตป่าสงวน/เขตหวงห้าม
  • ต้องมีหนังสืออนุญาตให้เข้าทำประโยชน์
  • ที่ดินเช่า → ต้องมีหนังสือแสดงสิทธิ + สัญญาเช่าทุกกรณี
  • ที่ดินถือครองมือเปล่า → ต้องไม่อยู่ในเขตป่าสงวน/สาธารณะประโยชน์ + มีลู่ทางได้กรรมสิทธิ์ในอนาคต

6.2 เงินกู้เพื่อพัฒนาความรู้

📌 หลักเกณฑ์
  • ผู้กู้ต้องเป็นลูกค้า ฉ.44 หรือ ฉ.45
  • ผู้เข้ารับการพัฒนาความรู้ = ลูกค้า หรือบุคคลในครอบครัว
  • ต้องมีคุณสมบัติตามระเบียบที่แหล่งให้ความรู้กำหนด
💰 วงเงินกู้:
  • ✅ มีรายได้สุทธิเพียงพอ ชำระหนี้ได้ภายในรอบปีการผลิต → กู้เป็น ค่าใช้จ่าย
  • ❌ มีรายได้สุทธิไม่เพียงพอ → กู้เป็น ค่าลงทุน (ใช้หลักประกันตามข้อบังคับ)

6.3 เงินกู้เพื่อพัฒนาคุณภาพชีวิตด้านการศึกษา

📌 หลักเกณฑ์
  • ผู้กู้ต้องเป็นลูกค้า ฉ.44 หรือ ฉ.45
  • ต้องมีคุณสมบัติตามระเบียบการคัดเลือกบุคคลเข้าศึกษา
  • ต้องเป็นผู้ที่ ไม่ได้รับเงินกู้ยืมเพื่อการศึกษาจากแหล่งอื่น หรือไม่อยู่ระหว่างขอกู้ยืม
💵 ค่าใช้จ่ายที่ครอบคลุม:
  • ค่าบำรุง / ค่าเล่าเรียน / ค่าลงทะเบียน / ค่าธรรมเนียม
  • ค่าอุปกรณ์การศึกษา / ค่าครองชีพ (ค่าเครื่องเขียน, ที่พัก, ค่าใช้จ่ายส่วนตัว)

6.4 เงินกู้เพื่อพัฒนาคุณภาพชีวิตด้านที่อยู่อาศัย

🏠 วัตถุประสงค์
  • ซื้อที่ดินเพื่อปลูกสร้างที่อยู่อาศัย
  • ซื้อที่ดินพร้อมสิ่งปลูกสร้าง
  • ซื้อที่อยู่อาศัย
  • สร้าง / ต่อเติม / ปรับปรุง / ซ่อมแซมที่อยู่อาศัย
💰 วงเงินกู้ขั้นสูง
  • 📉 รายได้ไม่ประจำ → กู้ไม่เกิน 3 เท่า ของรายได้หลังหักค่าใช้จ่ายต่อปี
  • 📈 รายได้ประจำและชำระหนี้ได้ทุกเดือน → กู้ไม่เกิน 30 เท่า ของรายได้ประจำต่อเดือนหลังหักค่าใช้จ่าย
  • ⚠️ รวมสัญญาเงินกู้พัฒนาคุณภาพชีวิตด้านที่อยู่อาศัย ต้องไม่เกิน 3 ล้านบาท

📌 หลักประกันเงินกู้ที่อยู่อาศัย

  • 🏠 อสังหาริมทรัพย์จำนอง / พันธบัตรรัฐบาล / เงินฝาก
  • 👥 ลูกค้าประจำสาขาหรือบุคคลอื่นค้ำประกัน อย่างน้อย 2 คน
  • 📝 กรณีเจตนาให้บุคคลในครอบครัวเป็นเจ้าของ → ต้องทำหนังสือรับรองรับผิดอย่างลูกหนี้ร่วม
  • 🛡️ กรณีกู้เพื่อตนเองแต่มีครอบครัวอยู่อาศัยร่วม → ต้องให้บุคคลดังกล่าวทำหนังสือค้ำประกันเพิ่ม
  • 🔥 แนะนำทำประกันวินาศภัยสิ่งปลูกสร้าง (จ่ายเป็นรายปี ห้ามบังคับจ่ายคราวเดียว)
  • ⚠️ ห้ามบังคับ ให้ลูกค้าสมัครสมาคมฌาปนกิจสงเคราะห์ผู้ฝากเงิน ธ.ก.ส. หรือทำประกันชีวิต (แนะนำได้เท่านั้น)

📌 ระยะเวลาชำระคืนที่อยู่อาศัย

⏰ ชำระคืนเสร็จไม่เกิน 15 ปี (มีเหตุพิเศษไม่เกิน 20 ปี)

6.5 เงินกู้เพื่อพัฒนาคุณภาพชีวิตด้านสุขภาพ/รักษาพยาบาล

📌 คุณสมบัติผู้ใช้เงินกู้ — ผู้กู้หรือบุคคลในครอบครัว
  • 🩺 จัดทำหลักประกันสุขภาพ (ซื้อประกันชีวิต/สุขภาพ)
  • 💉 ป้องกันตนเองจากโรคภัย (ตรวจสุขภาพประจำปี, ฉีดวัคซีน)
  • 🏥 รักษาพยาบาลในสถานพยาบาลรัฐหรือเอกชน

Mnemonics

📝 7 วัตถุประสงค์กู้เงิน (ข้อ 15 ฉ.44):
"จ่าย ทุน หนี้ รู้ ชีวิต ขาย โครงการ"
ค่าใช้จ่าย → ค่าลงทุน → ชำระหนี้ → พัฒนาความรู้ → พัฒนาคุณภาพชีวิต → รอขายผลิตผล → ตามโครงการ
📝 6 ขั้นตอนกระบวนการ:
"แจ้ง → สอบ → วิเคราะห์ → ตรวจ → อนุมัติ → จ่าย"
(จำแค่คำแรก: รับ-สอบ-วิเคราะห์-ตรวจ-อนุมัติ-แจ้ง)
📝 8 ปัจจัยวิเคราะห์ผู้กู้:
"ฐาน ชื่อ ประวัติ ความรู้ ต้องการ รายได้ ความสามารถ หลักประกัน"
📝 ไปทำแบบทดสอบ (ครบ 20+ ข้อ)
